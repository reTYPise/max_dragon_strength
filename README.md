# Max Dragon Strength

**Репозиторий:** [https://github.com/reTYPise/max_dragon_strength](https://github.com/reTYPise/max_dragon_strength)

Учебный проект: программа для вычисления **максимальной силы стаи драконов** при заданном общем количестве голов.

---

## Описание задачи (кейс)

Дано целое число **N** — общее количество голов у стаи драконов.  
Нужно разбить N на слагаемые (количество голов у каждого дракона) так, чтобы:

- у одного дракона было **не более 7 голов**;
- **сила стаи** (произведение количества голов всех драконов) была **максимальной**.

Вывести одно число — максимально возможную силу.

**Ограничения:** `0 < N < 100`. Ввод — одно число N (из файла/стандартного ввода), вывод — одно число (результат).

---

## Подробное описание действий при выполнении кейс-задачи

### 1. Анализ условия

- Уточнено: сила = произведение чисел голов по всем драконам.
- Ограничение «не более 7 голов у дракона» означает, что каждое слагаемое от 1 до 7.
- Требуется не просто разбить N в сумму, а выбрать такое разбиение, при котором произведение этих слагаемых максимально.

### 2. Математическая идея

- Задача сводится к: при фиксированной сумме N максимизировать произведение натуральных слагаемых.
- Известный факт: при фиксированной сумме произведение максимально, когда слагаемые по возможности равны **3** (и при необходимости используются **2**). Числа больше 4 выгодно разбивать (например, 5 → 2·3, 6 → 3·3, 7 → 3·4).
- Так как 2 и 3 не превышают 7, ограничение «≤ 7 голов» не меняет стратегию: оптимальное разбиение строится только из 2 и 3.

### 3. Алгоритм

- Если **N ≤ 3**: ответ 1, 2 или 3 соответственно (один дракон).
- Если **N ≥ 4**:
  - Считаем, сколько раз в N «влезает» 3: `count_threes = N // 3`, остаток `remainder = N % 3`.
  - **Остаток 0:** все головы группами по 3 → ответ `3^count_threes`.
  - **Остаток 1:** одна тройка «забирается» и вместе с 1 даёт 4 = 2·2; ответ `3^(count_threes-1) * 4`.
  - **Остаток 2:** одна группа из 2 голов; ответ `3^count_threes * 2`.

### 4. Реализация

- Написан модуль на Python с функцией `calculate_max_dragon_strength(n)` по описанному алгоритму.
- Ввод: число N из `sys.stdin` (удобно для тестов и проверки через перенаправление файла).
- Проверка диапазона `0 < N < 100`; при ошибке ввода выводятся понятные сообщения.
- Результат выводится одним числом в stdout.

### 5. Проверка

- Проверены граничные случаи: N = 1, 2, 3, 4, 5, 6, 7 и несколько больших N.
- Убедились, что ответы совпадают с ручным расчётом (например, N = 5 → 2·3 = 6, N = 6 → 3·3 = 9).

## Как запустить

**Требования:** Python 3.6+

```bash
# Из консоли (ввод числа с клавиатуры, затем Enter)
python max_dragon_strength.py

# Или с перенаправлением из файла (в файле одно число N)
python max_dragon_strength.py < input.txt
```

Пример: ввести `6` → программа выведет `9` (два дракона по 3 головы: 3·3 = 9).
